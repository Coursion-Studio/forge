What's inside

Every tool an agent needs.

Split, tab, and arrange panes per workspace. Sessions survive workspace switches — terminals keep running, chats keep streaming, pages stay loaded.

Terminal

Real PTY terminals that keep running when you switch workspaces. Launch Claude Code, Codex, Aider, or any CLI agent straight into your project — Forge detects what's installed.

  • Persistent sessions
  • Agent auto-detect
  • Login-shell PATH
  • Activity status

AI Chat

Talk to Anthropic or OpenAI models with your own keys, stored in the Keychain. Compare multiple agents side-by-side in the Compare pane.

  • BYO API keys
  • Streaming
  • Multi-agent compare
  • Keychain only

Browser

A real WKWebView next to your terminal — preview your dev server, read docs, test the thing the agent just built. Page state survives workspace switches.

  • Persistent pages
  • Dev-server preview
  • Per-leaf sessions

    Why Forge

    Native. Local. Yours.

    Cloud agent workspaces, minus the cloud.

    100% native SwiftUI

    No Electron. No web views pretending to be apps. Feels like macOS because it is macOS.

    Fully local

    No VMs, no containers, no accounts. Shell runs on your Mac, files live in your folders. AI is optional and runs on your own keys.

    Workspaces that persist

    Split panes, tab groups, per-workspace accents. Switch projects with ⌘1–⌘9 — everything keeps running.

    Auto-updates

    Sparkle-signed updates ship over the air. Apple-notarized DMGs, EdDSA-verified — no App Store middleman.
